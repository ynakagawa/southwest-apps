Southwest Apps

Embeddable Widget Suite — built as standalone JS bundles for any HTML page

📦 Each tab below builds a self-contained custom element bundle (flight-search-widget.js, login-widget.js, feedback-widget.js). Click "Show embed code" to see exactly what to drop into an EDS block or any HTML page.

Flight Search

The southwest.com search module: trip type, depart/arrive, dates, passengers, promo code, fare display, and low-fare-calendar option. Submitting redirects to a southwest.com results URL built from the form fields. 

<script src="https://YOUR_CDN_OR_HOST/flight-search-widget.js"></script>
<sw-flight-search></sw-flight-search>

<!-- Or as an EDS block: blocks/flight-search/flight-search.js
     loads the script and mounts the element automatically. -->

Login

A yellow "Log in" trigger button that opens a popover (account number/username, password, remember me, plus forgot-login and enroll links) anchored below the button, matching the southwest.com header. 

<script src="https://YOUR_CDN_OR_HOST/login-widget.js"></script>
<sw-login></sw-login>

<!-- Or as an EDS block: blocks/login/login.js
     loads the script and mounts the element automatically. -->

Feedback

A vertical "Feedback" tab fixed to the left edge of the viewport that expands into a panel: goal dropdown, accomplished yes/no, comments textarea, and a Contact Us link — matching the southwest.com feedback widget. 

<script src="https://YOUR_CDN_OR_HOST/feedback-widget.js"></script>
<sw-feedback></sw-feedback>

<!-- Or as an EDS block: blocks/feedback/feedback.js
     loads the script and mounts the element automatically. The tab and
     panel are position: fixed inside the widget's own shadow root, so
     one instance anywhere on the page is enough. -->
