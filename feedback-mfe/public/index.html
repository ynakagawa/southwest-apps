Feedback Widget

A self-contained <sw-feedback> web component built from a React app, meant to be embedded as a micro-frontend in an Adobe Edge Delivery Services (EDS) site.

  • feedback-widget.js — the built bundle. Load it with a <script> tag to register the custom element.
  • /examples/embed-demo.html — a live demo mounting the fixed feedback tab and panel on a bare page, the same way an EDS block would.
